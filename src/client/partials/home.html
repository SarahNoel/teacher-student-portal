Connect With your Students...


and keep them connected to each other.

Student Space provides a safe and secure place to connects students and teachers

Features

Safety for Students,
Safety for You

Chat and message in a secure environment. Only students with your access code can join your room. Language is filtered for students, but teachers can still see the original. All chat history is saved to provide safety for all parties.

Share Flashcards
and Resources

Students can create flashcards for themselves for any subjects. Teachers can create sets that are shared with all students in their classes.

Create And Share Games
with Your Class

Teachers can easily create Vocab Racer and Hangman games for their classes. They can also add and remove links to outside resourses for their students.

Students Chat Together

Students can direct message each other with the same safe filters applied. This provides a secure, monitored place for chatting, and doesn't exclude those without phones to text.


Get Started


Register a Teacher

Teachers create a unique code to allow students access. They also give a valid phone number to allow students to alert them when in distress. (Have no fear, this feature can be disabled completely or by student!)

Create Class Goodies

The teacher can now create games and flashcards, as well as add resources for the students to access. These can all be edited, deleted, or more created at any time.

Students Sign Up!

Now give that code to your students, and voila! You have a safe and secure Student Space for your class to enjoy.
